FTE is a font editor.

The Details

FTE is still in development. A large amount of features is either incomplete, unpolished, or outright unavailable. Keep that in mind when using the software for your own work.

Intuitive editor

Combining the best elements of type editors and vector graphics software.

Modifier system

Non-destructive geometry adjustments and artistic effects.

Open source

Written in TypeScript and freely available to the public.
